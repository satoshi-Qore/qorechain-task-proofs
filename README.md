# QoreChain Task Proofs

This repository collects public proof notes for QoreChain Eigenstate 2 tasks.

The purpose is to keep screenshots, links, dates, and task descriptions organized in one place instead of posting every proof separately on social media.

## How To Use

1. Complete a task.
2. Take a screenshot if the task does not generate a public proof link.
3. Add a short proof note under `eigenstate-2/`.
4. Upload the screenshot under `screenshots/`.
5. Submit the proof note URL in the QoreChain task review form.

## Notes

- Do not upload private keys, seed phrases, wallet credentials, or personal documents.
- Use screenshots only when the task does not provide a direct proof link.
- Keep proof notes factual and short.
- Do not claim that a task is approved until the QoreChain review system marks it as approved.
